# Raptor Runner v1.0.4 — Coins, Shop, Cosmetics, and a Big Polish Pass

This one's been cooking for a while. Since v1.0.3 the game's grown a full economy layer, a cosmetics shop, a whole pile of achievements, and a bunch of sky, audio, and rendering polish that should make every run feel tighter than before. Also — the mobile build is real now.

## What's new

**Coins + shop**
- A small ribbon of coins appears on every flower-field breather. Grab them with a short hop to fund the shop.
- Each pickup plays an upward chromatic pitch chain so chaining coins sounds like a little treasure-chest flourish.
- Coin hitboxes are now polygon-accurate and the icons hover just above tall cacti so they stay grabbable.

**Cosmetics**
- New Shop section in the menu with its own full-screen overlay: pause the game, spend your coins, dress up the raptor.
- Three **score-unlock classics** earned through gameplay: Party Hat at 100 points, Thug Glasses at 250, and Bow Tie at 500. Each one bursts confetti over the raptor the moment you cross the threshold and equips itself automatically.
- Shop inventory: cowboy hat, top hat, wizard hat, pirate tricorn, sombrero, tiara, monocle, eye patch, gold chain, bandana, and a premium jewelled crown.
- Start-screen raptor wears whatever you've got equipped — preview your outfit before you press start.
- Per-slot equip menu inside the pause menu so swapping is one click.

**Achievements**
- Four new shop + economy achievements: first purchase, shop-cleaned-out, "Jurassic Runway" (one cosmetic equipped in every slot), and Scrooge McRaptor (1,000 lifetime coins).
- Dedicated unlock stinger that plays over the toast — clean, short glockenspiel.

**Sky**
- Full twilight chain extended to 14 bands so dusk transitions through golden hour → sunset → blue hour → night rather than the old abrupt fade.
- Moon terminator fixed — you no longer see two "full moon" phases per cycle (turns out my ellipse math was using the wrong axis).

**Mobile**
- Raptor Runner now runs natively on iOS and Android via Capacitor. Haptics on jump / thunder / death, safe-area insets, orientation lock, splash, and durable save mirroring to the native preferences store so your progress survives WebView eviction.
- Gamepad support across the whole menu system — Switch Pro, PlayStation, Xbox, and generic 8BitDo controllers all navigate cleanly.

**Desktop**
- Steam-ready Electron build with Steamworks achievements wiring. The app-id gate keeps Spacewar out of non-Steam distributions.
- Cross-platform build pipeline (macOS / Windows / Linux) via electron-builder, plus itch.io butler publish scripts.

## Polish + perf

- Confetti bursts no longer stutter the frame they spawn on — 70-piece bursts used to pay for 140 full context-state snapshots per frame via save/restore; now it's zero save/restores with a single composed transform per particle.
- Cosmetic unlocks no longer spike the frame they fire on — up to five synchronous localStorage writes are now batched and flushed during idle time, with a sync flush on page-hide so nothing is lost.
- Lightning strikes no longer drop the frame rate during storms: the bolt is baked once to a bounding-box-sized offscreen canvas and blitted with a single drawImage.
- Coin sprites are cached once instead of being re-drawn from gradients every frame.
- PWA service worker icon dedup bug fixed — cold-opening the installed PWA no longer returns to a stuck-on-splash state.

## Under the hood

- Strict CSP — dropped `unsafe-inline` for scripts entirely, ~1700 lines of inline logic moved into `src/ui.ts` as a proper module, SHA-256 hashes cover the small remainder.
- Audio subsystem: pre-decode + graph warmup on first user gesture so the first jump / thunder / death sound no longer "plops".
- Rare-event audio (UFO / Santa / meteor) now stops cleanly if you die mid-event instead of looping into the game-over screen.

## Known stuff

- The cosmetic lineup ended up without any back-slot wings in the end — the earlier angel/demon/butterfly wing sprites are gone. (Side-view silhouette + shoulder-attachment math was never going to land clean. Cut the whole slot.)
- No mobile store listing yet — the LLC is freshly formed and the Apple Developer Program enrollment is in flight. Mobile builds currently run only via dev install.

Next up: on-device QA pass, store listings, and start shipping actual builds to places players can download them.
