# Raptor Runner v1.0.3 — Windows desktop build unblocked

Quick Windows-only fix release.

## What was broken

v1.0.2 shipped with a broken Windows desktop build. The Electron + Capacitor build scripts used shell-prefix env-var syntax — `VITE_TARGET=electron vite build` — which works fine on macOS and Linux but is POSIX-only. Windows `cmd` reads `VITE_TARGET=electron` as an unknown command and dies before Vite even starts.

Mac and Linux players were fine. Windows players who downloaded v1.0.2 couldn't build the desktop target from source at all.

## The fix

Wrapped every target-env-var invocation in **cross-env**, which normalises the assignment syntax across POSIX shells and Windows cmd. Scripts look identical on every platform now.

No code changes to the game itself — just the build tooling. If you're playing the hosted web version or the already-shipped Mac / Linux binaries, v1.0.3 has nothing new for you.

## Action for you

If you tried to build the Windows desktop target from source and hit the `VITE_TARGET` error, pull v1.0.3 and rerun the build. The prebuilt Windows binaries on itch.io were re-published against v1.0.3 too, so grab a fresh download if you were on the broken one.
