# Raptor Runner v1.0.2 — Icon refresh + UI font swap

Small visual polish release.

## What changed

**New icon.** The launch icon had the raptor sitting a bit small inside the rounded black border. Enlarged and recentered the sprite so the tail extends off the left edge while the body and head stay fully inside the frame — reads better at favicon size and on home-screen installs. All the derivative sizes (192, 512, apple-touch, rounded variants) regenerated from the new source.

**New OG image.** The share / link-preview card also got updated to the new framing so when the game's URL gets dropped into chats / social the thumbnail matches the app icon.

**UI font swap.** Moved the menu / score display / overlay chrome off the system default and onto **Anonymous Pro**. Slightly more "retro terminal" feel without being hard-to-read, and it matches the pixel-art aesthetic of the gameplay a little better.

## Housekeeping

Removed a stray `launch.json` that had ended up in `public/assets/fonts/` and didn't belong in the deployed bundle.

## No gameplay changes

Nothing about the game loop, scoring, achievements, or weather moved. If you're playing fine on v1.0.1, v1.0.2 is just cosmetic.

— Rico

> Editorial note from a later release: the Anonymous Pro switch got reverted in a subsequent version — the font loaded noticeably slower on cold visits than the system stack, and the readability win wasn't worth the FOUT. But it shipped in v1.0.2.
